# AtomBadgeNotification

> Atom Element: SUI badgeNotification

## Installation

```sh
npm install @schibstedspain/sui-atom-badge-notification --save
```

## Usage

```js
import BadgeNotification from '@schibstedspain/sui-atom-badge-notification'

render {
  return (
    <BadgeNotification />

}

```
